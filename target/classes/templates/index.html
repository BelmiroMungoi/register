Olá, Seja Bem Vindo ao Sistema

Esse sistema foi criado com o intuito de treinar a arquitetura MVC usando o Spring Framework. A Aplicacão, resumindo, é um sistema de Gestão de Funcionários. Onde é possivel registar um funcionário, editar, pesquisar e deletar(somente o admin pode fazer isso), pode tambem adicionar um currículo para o funcionário e baixar caso seja necessário. Pode adicionar enderecos do funcionário ao clicar no nome funcionário.

Pode também baixar o relatório de funcionários, pode escolher se quer baixar somente o relatório de funcionários do sexo Masculino ou Feminino, ao clicar na opcão de escolha de sexo, pode baixar relatórios de usuários com um determinado nome.
